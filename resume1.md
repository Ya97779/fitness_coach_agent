## 💼 项目经验

### FitCoach AI 智能私教系统（个人项目）

**项目简介**：基于大模型的智能健身教练系统，模拟专业营养师和健身教练的协作流程，为用户提供个性化的饮食计划和运动指导。

**技术栈**：Python / FastAPI / LangGraph / ChromaDB / Streamlit / SQLite

**核心职责**：
- 设计并实现多 Agent 协作架构，包含闲聊、营养师、健身教练、专家评审四个专用 Agent
- 基于 LangGraph 实现动态路由系统，根据用户输入自动分发到合适的 Agent
- 集成 RAG 系统，基于 ChromaDB 向量数据库实现医学健身知识检索
- 封装天行数据食物营养 API，实现食物热量查询和摄入记录功能
- 开发类 ChatGPT 风格的 Web 界面，支持流式输出和实时对话

**技术亮点**：
- 创新性地将 Multi-Agent 架构应用于健康管理领域，实现专业化分工
- 使用 ReAct 模式实现 Agent 工具调用，提升系统智能化水平
- 通过异步编程和流式输出优化用户体验
